# Спецификация: Три новые страницы Scandic School

**Дата:** 2026-04-23
**Статус:** Draft

## Обзор

Создать три новые страницы на основе контента из презентации школы:
1. `/structure` — Структура школы (управление, учебная программа, инфраструктура)
2. `/community` — Учителя, дети, родители (философия взаимоотношений)
3. `/programs` — Академическая программа (предметные области, STEM)

Все страницы — на трёх языках (ru, en, kk). Визуальный подход B: переиспользуем существующие паттерны (hero + чередующиеся секции + карточки + ScrollReveal).

## Страница 1: `/structure` — Структура школы

### Hero секция
- Тёмно-зелёный градиент (`bg-gradient-to-br from-secondary via-secondary-800 to-secondary-900`)
- `ParticleBackground`
- Заголовок: "Структура школы"
- Подзаголовок: "2-уровневое профессиональное управление. Международная учебная программа"
- Жёлтая акцентная линия внизу

### Секция "Управление школой"
- Белый фон
- Стандартный section header с label "Управление" + заголовок "Как устроена наша школа"
- Два визуальных уровня:

**Верхний уровень — "Операционное управление":**
Три карточки в ряд (стандартный паттерн `rounded-2xl border border-gray-100 bg-white hover:border-secondary/30`):
  - CEO школы — иконка `Crown`, краткое описание роли
  - Менеджмент школы — иконка `Briefcase`, описание
  - Попечительский совет школы — иконка `Shield`, описание

**Нижний уровень — "Общее управление / координация":**
Одна карточка по центру (шире остальных):
  - PTA (Parent-Teacher Association) — иконка `Users`, описание роли

Между уровнями — тонкий разделитель с пунктирной линией и label "координация".

### Секция "Учебная программа"
- Светлый градиентный фон (`bg-gradient-to-br from-white via-gray-50/50 to-secondary/[0.02]`)
- Section header: label "Программа" + заголовок "Учебная программа по ступеням"
- Три колонки-карточки с цветным акцентом сверху (4px border-top):

1. **Начальная школа** (акцент: primary/жёлтый)
   - Подзаголовок: "Фундамент образования"
   - Содержание: критическое мышление, индивидуальность, навыки через игру и исследования
   - Теги: "IB PYP", "Трилингвальная среда"

2. **Средняя и старшая школа** (акцент: secondary/зелёный)
   - Подзаголовок: "Путь к академическому лидерству"
   - Содержание: углублённое изучение STEM, подготовка к международным экзаменам
   - Теги: "IB MYP & DP", "Научные проекты", "Гостевые лекции"

3. **Внеучебное развитие** (акцент: accent/оранжевый)
   - Подзаголовок: "Развитие практических навыков"
   - Содержание: самостоятельность через проекты и дебаты, ролевые опыты
   - Теги: "Контроль ДЗ", "Кружки и секции", "Работа с экспертами"

### Секция "Инфраструктура"
- Тёмно-зелёный фон (`bg-secondary`)
- Section header (белый текст): label "Инфраструктура" + заголовок "Блоки школы"
- 4 карточки в ряд (паттерн stat card на тёмном: `bg-white/[0.06] backdrop-blur-sm border border-white/10`):
  - Аудитории высшего класса — иконка `School`
  - Родительский хаб / концептуальный ресторан — иконка `Coffee`
  - Лаборатория STEM — иконка `FlaskConical`
  - Лаборатория физики и экспериментов — иконка `Microscope`

### CTA секция
- Светлый градиент (`bg-gradient-to-br from-primary/10 via-primary/5 to-white`)
- 3 карточки-ссылки (паттерн About CTA): Академическая программа → `/programs`, О нас → `/about`, Подать заявку → `/application`

- Крупный текст-цитата по центру (стиль blockquote как на About):

2. **Дети** — иконка `Heart`

3. **Родители** — иконка `Home`
   - Цветной акцент сверху: accent/оранжевый

- Горизонтальная полоса со ссылками: О нас → `/about`, Коллектив → `/staff`, Структура → `/structure`

## Страница 3: `/programs` — Академическая программа

### Hero секция
- Тёмно-зелёный градиент + `ParticleBackground`
- Заголовок: "Академическая программа Scandic School"
- Подзаголовок: "Междисциплинарный подход: фокус на STEM и развитие личности"

### Секция "Предметные области"
- Белый фон
- Section header: label "Предметы" + заголовок "Наши предметные области"
- Четыре крупных блока, чередующиеся по расположению (как на Partners — alternating left/right layout):

1. **Естественные науки** — цветной акцент слева: зелёный
   - Иконка: `FlaskConical`
   - Описание: физика, химия и биология через исследование, эксперименты и практическую деятельность. Наука не по учебнику, а через реальные опыты и проекты. Обучение ведётся на английском языке, что позволяет одновременно осваивать предмет и развивать язык в естественной среде.
   - Пункты: "Экспериментальные модули", "Экологические проекты"

2. **Математика** — цветной акцент справа: терракотовый/оранжевый
   - Иконка: `Sigma` (или текст "Σ")
   - Описание: Calculus, арифметика, алгебра, геометрия — выдающийся уровень развития логики и аналитического аппарата высшего уровня.

3. **Искусство и История** — цветной акцент слева: тёмно-бирюзовый
   - Иконка: `Palette`
   - Описание: визуальное искусство, музыка и драма как инструменты самовыражения.

4. **Языковая программа** — цветной акцент справа: primary/жёлтый
   - Иконка: `Languages`
   - Описание: трилингвальная среда (казахский, русский, английский), изучение языков через контекст и практику.

### Секция-тизер "Структура школы"
- Жёлтый фон (`bg-primary`)
- Крупный текст в тёмно-зелёном: "Узнайте, как устроена наша школа — от управления до инфраструктуры"
- GlowButton или стилизованная ссылка → `/structure`

### Секция "STEM фокус"
- Тёмно-зелёный фон (`bg-secondary`)
- Section header (белый текст): label "STEM" + заголовок "Междисциплинарный подход"
- Описание: как предметы связаны между собой, фокус на практическом применении знаний
- Иконка или визуальный элемент STEM

### CTA секция
- 3 карточки-ссылки: Подать заявку → `/application`, О нас → `/about`, Контакты → `/contact`

## Локализация

Ключи для каждой страницы следуют паттерну существующих (`about`, `partners`):

```json
{
  "structure": {
    "heroLabel": "...",
    "title": "...",
    "subtitle": "...",
    "management": { ... },
    "curriculum": { ... },
    "infrastructure": { ... }
  },
  "community": {
    "heroLabel": "...",
    "title": "...",
    "subtitle": "...",
    "philosophy": { ... },
    "sides": { ... }
  },
  "programs": {
    "heroLabel": "...",
    "title": "...",
    "subtitle": "...",
    "subjects": { ... },
    "stem": { ... }
  }
}
```

Полные тексты переводов будут написаны при реализации. Русский — из контента слайдов. Английский и казахский — перевод русского.
